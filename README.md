The program includes a list of students to keep track of a school's information.

The program starts on the main.py file, where the menu function is activated. There, you will find the following options:

1. Add student: Function where you will find the option to add a new student in case of a new admission.
2. Read student's list: Showcases all of the students. Including their status.
3. Update student: Gives the option to update a student depending of their Student ID.
4. Search for a student: Searchs for a specific student through their ID.
5. Delete student: Deletes a student through their ID.
6. Save data as CSV: Downloads a CSV file using all the student's information.
7. Load CSV: Uploads a CSV file to a dictionary with all the student's information.
8. Exit

Example:
If you start the program it will show you the main menu and you will have the possibility to choose one of the options. Then, each function will run accordingly.
